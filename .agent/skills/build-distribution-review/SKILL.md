- [ ] `swift-tools-version: 6.2` — verify the toolchain is available on CI and dev machines.

## 2. DMG Packaging Script

### 2.1 `scripts/package_dmg.sh`
- [ ] Verify the script:
  1. Generates brand assets (optional — runs `generate_brand_assets.swift`).
  2. Builds release binary for `arm64-apple-macosx`.
  3. Creates `.app` bundle structure with `Info.plist`.
  4. Copies binary and resources to `.app/Contents/MacOS/` and `.app/Contents/Resources/`.
  5. Ad-hoc signs the app.
  6. Creates compressed DMG via `hdiutil`.

### 2.2 Info.plist Completeness
- [ ] Required keys:
  - `CFBundleIdentifier` — should be a proper reverse-domain (e.g., `com.eslamfaisal.cleanmymac`).
  - `CFBundleName` — "CleanMyMac".
  - `CFBundleShortVersionString` — matches release tag.
  - `CFBundleVersion` — build number.
  - `CFBundleExecutable` — binary name.
  - `CFBundleIconFile` — references `AppIcon.icns`.
  - `LSMinimumSystemVersion` — "14.0".
  - `NSHighResolutionCapable` — `true`.
- [ ] Optional privacy keys:
  - `NSDesktopFolderUsageDescription` — explain Desktop access.
  - `NSDownloadsFolderUsageDescription` — explain Downloads access.
  - No Full Disk Access usage description exists in Info.plist (it's a system-level permission, not TCC-prompted).

### 2.3 Code Signing
- [ ] Currently **ad-hoc signed** (`codesign -s -`). This means:
  - ⚠️ Gatekeeper blocks on first launch.
  - ⚠️ No notarization = macOS Sequoia may quarantine.
  - **Recommendation**: Set up Developer ID signing with `codesign -s "Developer ID Application: ..."`.
  - **Recommendation**: Notarize with `xcrun notarytool submit`.

## 6. Verification Commands

```bash
# Full build + test pipeline
swift build --scratch-path .build && swift test --scratch-path .build

# Package DMG
chmod +x scripts/package_dmg.sh && ./scripts/package_dmg.sh

# Verify DMG contents
hdiutil attach dist/CleanMyMac.dmg -mountpoint /tmp/cleanmymac-verify
ls -la /tmp/cleanmymac-verify/
codesign -dvvv /tmp/cleanmymac-verify/CleanMyMac.app
hdiutil detach /tmp/cleanmymac-verify

# Check signing
codesign --verify --deep --strict dist/CleanMyMac.dmg
```
